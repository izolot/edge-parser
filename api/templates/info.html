1. Получить события по времени

GET /api/events?

start - дата начала событий формат (%Y-%m-%dT%H:%M:%S)

end - дата окончания событий формат (%Y-%m-%dT%H:%M:%S)

uuid - номер камеры (36 знаков)

Пример:

api/events?start=2019-06-15T00:00:00&end=2019-06-15T23:59:00&uuid=bd6a0672-b72d-42d3-a54e-e1f9e7467a93

2.Получить последний excel файл по uuid камеры

GET /api/excel?

uuid - номер камеры (36 знаков)

Пример:

api/excel?uuid=bd6a0672-b72d-42d3-a54e-e1f9e7467a93

SUCCES -> excel последних выгруженных событий из прошлого запроса api/events?

ERROR -> {'message': 'File is not found'}
